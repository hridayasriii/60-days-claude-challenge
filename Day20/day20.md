# Day 20 - Face Puzzle Game

## Project
Face Puzzle Game using HTML, CSS and JavaScript.

## Features
- Webcam Access
- Capture Photo
- 3x3, 4x4 and 5x5 Difficulty
- Drag and Drop Puzzle
- Touch Support
- Timer
- Move Counter
- Correct Pieces Counter
- Results Screen
- Leaderboard using LocalStorage
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API
- getUserMedia API
- LocalStorage

## Learnings
- Camera integration using getUserMedia()
- Canvas image processing
- Drag and Drop API
- Touch Events
- DOM Manipulation
- LocalStorage
- Timer implementation
- Responsive UI
